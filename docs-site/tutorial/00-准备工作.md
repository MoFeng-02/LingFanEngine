# 准备工作

完整教程将带你从零开始，逐步掌握灵泛引擎的所有功能。学完之后，你将能独立开发并发布一款视觉小说游戏。

## 前置要求

- 已阅读[快速入门](../guide/00-什么是灵泛引擎)，了解引擎基本概念
- 已按[安装指南](../guide/01-安装与创建项目)创建了一个可运行的项目
- 有基本的文本编辑能力（任何编辑器均可，SDK 内置编辑器提供语法高亮）

## 教程约定

本教程中的代码示例可以用两种方式运行：

### 方式一：复制到项目运行

将 `.story` 代码复制到你项目的 `Stories/` 目录，在标题场景中添加按钮导航过去。

### 方式二：使用配套示例

每章都有配套的可运行示例文件，位于 `docs-site/examples/` 目录：

```
examples/
├── 01-first-scene.story       # 第 01 章
├── 02-dialog.story            # 第 02 章
├── ...
└── final-game/                # 最终完整作品
    └── final-game.story
```

## 坐标系统回顾

引擎使用**百分比坐标**，相对于设计分辨率（默认 1920×1080）：

```
(0%, 0%) ────────────── (100%, 0%)
  │                         │
  │      (50%, 50%)         │
  │        屏幕中央          │
  │                         │
(0%, 100%) ──────────── (100%, 100%)
```

所有 UI 元素的位置都用 `x=` 和 `y=` 指定百分比。
